How Thermal Printing Works

The app sends receipt data directly to your thermal printer over your local network. Your printer must:

  • Be a network-capable ESC/POS thermal printer (e.g., Epson TM series, Star TSP series, BIXOLON)
  • Be connected to your local network via Ethernet or Wi-Fi
  • Have a static IP address configured (set on the printer or via your router)
  • Accept connections on port 9100 (standard for most models)

Compatible Printers & Setup Tips
Epson TM Series
  • TM-T20III, TM-T88VI, TM-T70II
  • Connect via Ethernet cable
  • Use EpsonNet Config to assign a static IP
  • Default port: 9100
Star TSP Series
  • TSP143, TSP654, mC-Print3
  • Use Star Utility to find the IP
  • Enable "RAW" mode on port 9100
  • Wi-Fi or Ethernet supported
BIXOLON / Others
  • SRP-350, SRP-Q300
  • Print a self-test page to find the IP
  • Port is usually 9100
  • Must support ESC/POS commands

Tip: To find your printer's IP, hold the paper-feed button while powering on — most printers print a configuration page showing the IP address.
